# エリアタグ機能定義書

**バージョン:** 1.1
**作成日:** 2026-06-27
**更新日:** 2026-06-28

---

## 1. 機能概要

投稿に地域・エリアを示すタグ（例：北海道、京都、沖縄）を付与し、タグ別に投稿を絞り込み表示する機能。
エリアタグは投稿 1 件につき 1 つ設定する。

また、投稿には都道府県（`prefecture`）を別途ドロップダウンで設定できる。都道府県は 47 都道府県 ＋ 海外から選択する構造化データとして管理し、旅行レポートの集計および総合検索のエリアタブ絞り込みに使用する。

---

## 2. ユーザーストーリー

| ID | ストーリー |
|----|-----------|
| US-01 | ユーザーとして、投稿にエリアタグを付けて地域情報を伝えたい |
| US-02 | ユーザーとして、特定エリアの投稿だけを一覧で見たい |
| US-03 | ユーザーとして、投稿カードでエリアタグをクリックしてそのエリアの投稿一覧に遷移したい |
| US-04 | ユーザーとして、都道府県ドロップダウンで投稿をエリア別に絞り込みたい |
| US-05 | ユーザーとして、ホームの「エリアから探す」から都道府県別検索に素早くアクセスしたい |

---

## 3. 機能詳細

### 3.1 タグ付け（エリアタグ・自由記述）

- 投稿作成・編集フォームで入力する
- 入力方式：テキスト入力（例：嵐山、那覇、新宿）
- 1 投稿につき 1 エリアタグ
- タグの形式：自由文字列。スポット近くの地名を記述するのを想定

### 3.2 都道府県選択（prefecture・構造化データ）

- 投稿作成・編集フォームで **47 都道府県 ＋ 海外** のドロップダウンから選択する（任意）
- 選択肢は北海道から沖縄県の順＋海外で固定（マスターデータとして定義）
- 投稿が存在する都道府県のみ検索のエリアタブに表示される
- エリアタグとは独立したフィールド（`prefecture`）で管理する

### 3.3 タグ絞り込み表示（エリアタグ起点）

- `/tags/[tag]` でエリアタグ別の投稿一覧を表示する
- 表示順：新着順（createdAt DESC）
- 投稿カードのエリアタグをクリックすると `/tags/[tag]` に遷移する

### 3.4 都道府県別絞り込み（検索エリアタブ）

- 総合検索（`/search`）の「エリア」タブで都道府県ボタンを一覧表示する
- 投稿が 1 件以上存在する都道府県のみ表示（動的）
- 都道府県ボタンをクリックすると該当する投稿を絞り込み表示
- カテゴリフィルター・並び順（新着 / 人気）は旅スポットタブと共通で機能する
- 再クリックで選択解除

### 3.5 ホームからの導線

- ホーム（`/`）の「エリアから探す」セクションに「都道府県で探す →」リンクを設置し、検索のエリアタブ（`/search?tab=area`）に直接遷移する

### 3.6 タグの管理方式

- エリアタグは Post テーブルの `areaTag` カラムにシンプルな文字列として保存する
- 都道府県は `prefecture` カラムに保存する（値域：47 都道府県名 ＋「海外」、または NULL）
- 専用のタグマスターテーブルは持たない
- 未定：エリアタグの正規化（大文字小文字・ひらがな/カタカナ統一）は実装時に検討

---

## 4. 使用 API

| Method | エンドポイント | 説明 | 認証 |
|--------|--------------|------|------|
| GET | `/api/posts?areaTag=[tag]` | エリアタグ別投稿一覧取得 | 不要 |
| GET | `/api/posts/explore?prefecture=[pref]` | 都道府県別投稿一覧取得 | 不要 |

---

## 5. バリデーション仕様

| 項目 | フロントエンド | バックエンド |
|------|-------------|------------|
| エリアタグ（areaTag） | 必須 | 必須・1〜50文字 |
| 都道府県（prefecture） | 任意・ドロップダウン選択（47都道府県＋海外） | 任意・許可値のみ受け付ける（10文字以内） |

---

## 6. 制約・前提条件

- エリアタグは 1 投稿に 1 つのみ
- タグマスターは持たないため、自由入力となる（タイポが混在する可能性がある）
- タグのサジェスト機能は未定
- 都道府県は任意項目のため NULL 許容。未設定の投稿はエリアタブに表示されない
- 検索エリアタブは投稿が存在する都道府県のみ動的に表示する

---

## 7. 対象外仕様

| 機能 | 理由 |
|------|------|
| 複数タグの設定 | スコープ外（1 投稿 1 タグに限定） |
| タグマスター管理 | スコープ外 |
| タグのランキング・人気順表示 | スコープ外 |
