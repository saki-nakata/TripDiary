# コメント機能定義書

**バージョン:** 1.0
**作成日:** 2026-06-27
**更新日:** 2026-06-28

---

## 1. 機能概要

投稿に対してテキストでコメントを追加する機能。
コメントは投稿詳細画面で一覧表示され、自分のコメントのみ削除できる。

---

## 2. ユーザーストーリー

| ID | ストーリー |
|----|-----------|
| US-01 | ユーザーとして、旅行スポットの投稿に質問やおすすめ情報をコメントしたい |
| US-02 | ユーザーとして、投稿のコメント一覧を時系列で確認したい |
| US-03 | ユーザーとして、各投稿のコメント数をフィードで確認したい |
| US-04 | ユーザーとして、自分のコメントを削除したい |

---

## 3. 機能詳細

### 3.1 コメント投稿

- 投稿詳細画面のコメント入力欄から投稿する
- `POST /api/posts/[id]/comments` で送信する
- 投稿後、コメント一覧に即時反映する

### 3.2 コメント一覧表示

- 投稿詳細画面（`/posts/[id]`）で表示する
- 並び順：古い順（createdAt ASC）
- 各コメントに表示する情報：
  - コメント投稿者のアバター・ニックネーム
  - コメント本文
  - 投稿日時
  - 削除ボタン（自分のコメントのみ表示）

### 3.3 コメント数表示

- フィード画面の投稿カードにコメント数を表示する
- コメント数は `comments` テーブルの件数を都度集計する（非正規化カウンタは未定）

### 3.4 コメント削除

- 自分のコメントのみ削除可能
- `DELETE /api/comments/[id]` で削除する
- 削除確認は未定（実装時に検討）

---

## 4. 使用 API

| Method | エンドポイント | 説明 | 認証 |
|--------|--------------|------|------|
| GET | `/api/posts/[id]/comments` | コメント一覧取得 | 不要 |
| POST | `/api/posts/[id]/comments` | コメント投稿 | 必要 |
| DELETE | `/api/comments/[id]` | コメント削除 | 必要（本人のみ） |

**レスポンス例（コメント一件）**

```json
{
  "id": "cm_xxx",
  "body": "とても参考になりました！",
  "createdAt": "2026-06-27T10:00:00.000Z",
  "author": {
    "id": "u_xxx",
    "nickname": "田中花子",
    "image": "https://res.cloudinary.com/..."
  }
}
```

---

## 5. バリデーション仕様

| 項目 | フロントエンド | バックエンド |
|------|-------------|------------|
| コメント本文 | 必須・1〜2000文字・文字数カウンター表示 | 必須・1〜2000文字 |
| 未認証ユーザーのコメント投稿 | - | 401 を返す |
| 他人のコメント削除 | - | 403 を返す |

---

## 6. 制約・前提条件

- コメントはログインユーザーのみ投稿・削除できる
- コメントの最大文字数は2000文字
- コメントへの返信（ネスト構造）は対象外

---

## 7. 対象外仕様

| 機能 | 理由 |
|------|------|
| コメントへの返信（スレッド） | スコープ外 |
| コメントの編集 | スコープ外 |
