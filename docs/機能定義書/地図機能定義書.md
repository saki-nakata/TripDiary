# 地図機能定義書

**バージョン:** 1.1
**作成日:** 2026-06-27
**更新日:** 2026-06-30

---

## 1. 機能概要

投稿した旅行スポットの位置情報を地図上に表示する機能。
Leaflet + OpenStreetMap を使用し、投稿詳細画面にスポットの位置をピンで表示する。

---

## 2. ユーザーストーリー

| ID | ストーリー |
|----|-----------|
| US-01 | ユーザーとして、投稿詳細画面でスポットの場所を地図で確認したい |
| US-02 | ユーザーとして、投稿作成時にスポットの位置（緯度/経度）を設定したい |

---

## 3. 機能詳細

### 3.1 地図表示（投稿詳細）

- 投稿の `lat` / `lng` が設定されている場合のみ地図を表示する（地図拡大ボタンで地図であることが自明なため「🗺️ 地図」の見出しラベルは表示しない）
- 未設定の場合は地図エリア自体を非表示にする
- Leaflet + OpenStreetMap を使用してインタラクティブな地図を表示する
- スポットの位置にピンマーカーを表示する
- 初期ズームレベル：14
- 地図右上の「🗺️地図拡大表示🔍」ボタンから、画面全体を覆うモーダル表示（`w-full max-w-4xl h-[80vh]`）で拡大した地図を確認できる。ESCキーまたは背景クリック、右上の✕ボタンで閉じる

### 3.2 位置情報の設定（投稿作成・編集）

- 入力方式：地図上クリックでピンを立てる（案 B 採用）
- 位置情報の設定は任意とする（未設定でも投稿可能）
- クリック後、再クリックでピンを上書き移動できる
- ピン横の ✕ ボタンでピンと住所表示をリセットできる

### 3.3 逆ジオコーディング（住所表示）

- ピン設置後、Nominatim API で日本語住所を自動取得して地図の**下**に表示する
- エンドポイント：`https://nominatim.openstreetmap.org/reverse?lat={lat}&lon={lng}&format=json&accept-language=ja`
- 取得中は「取得中…」を表示し、取得完了後に `display_name` を表示する（例: `📍 千代田区, 東京都, 日本`）
- 取得失敗時は座標（`📍 lat, lng`）にフォールバックする
- 住所は表示用途のみ。DB には座標（lat/lng）のみ保存する

### 3.4 ライブラリ構成

| 項目 | 内容 |
|------|------|
| 地図ライブラリ | Leaflet 1.9.4（Next.js では `react-leaflet` + 動的 import） |
| タイル | OpenStreetMap（APIキー不要・無料） |
| 逆ジオコーディング | Nominatim（APIキー不要・無料） |

---

## 4. 使用 API

| Method | エンドポイント | 説明 | 認証 |
|--------|--------------|------|------|
| （外部）| Leaflet + OpenStreetMap | 地図タイル表示 | 不要 |
| （外部）| Nominatim `/reverse` | 逆ジオコーディング（座標→日本語住所） | 不要 |

※ サーバーサイドの Route Handler は地図機能専用には設けない。座標データは投稿 API（`POST /api/posts`）で一緒に送受信する。

---

## 5. バリデーション仕様

| 項目 | 仕様 |
|------|------|
| 緯度 | 任意・-90 〜 90 の範囲（`postSchema`の`lat`にZodバリデーション実装済み） |
| 経度 | 任意・-180 〜 180 の範囲（`postSchema`の`lng`にZodバリデーション実装済み） |

---

## 6. 制約・前提条件

- Leaflet + OpenStreetMap を使用する（APIキー・環境変数不要）
- 位置情報の設定は任意とする（未設定でも投稿可能）
- 地図クリックでピンを設置し、再クリックで上書き移動、✕ ボタンでリセットできる
- 住所はクライアント側の表示のみ。DB には lat/lng のみ保存する
- 地図表示はクライアントサイドコンポーネントとして実装する（`'use client'` が必要）

---

## 7. 対象外仕様

| 機能 | 理由 |
|------|------|
| 複数スポットの地図一覧表示 | 未定（実装時に検討） |
| ルート表示 | スコープ外 |
| GPS による現在地取得 | スコープ外 |
| オフライン地図 | スコープ外 |
