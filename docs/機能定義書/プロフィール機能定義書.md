# プロフィール機能定義書

**バージョン:** 1.0
**作成日:** 2026-06-27

---

## 1. 機能概要

ユーザーのプロフィール情報を表示・編集する機能。
プロフィール画面（公開）では投稿一覧・フォロワー/フォロー中数を確認できる。
訪問済み・行きたいリストはマイページ（非公開）で管理する。
プロフィール画像は Cloudinary にアップロードして管理する。

---

## 2. ユーザーストーリー

| ID | ストーリー |
|----|-----------|
| US-01 | ユーザーとして、他ユーザーのプロフィール（名前・アイコン・自己紹介・投稿一覧）を確認したい |
| US-02 | ユーザーとして、自分の名前・自己紹介を編集したい |
| US-03 | ユーザーとして、プロフィール画像をアップロード・変更したい |
| US-04 | ユーザーとして、マイページで自分の訪問済み / 行きたいリストを確認したい |
| US-05 | ユーザーとして、プロフィール画面でフォロワー数・フォロー中数を確認したい |

---

## 3. 機能詳細

### 3.1 プロフィール表示

- URL：`/users/[id]`
- 表示情報：
  - プロフィール画像（未設定時はデフォルトアイコン）
  - 名前（name）
  - 自己紹介（bio）
  - TabiScore（旅スコア・ランク）
  - 投稿数 / フォロワー数 / フォロー中数
  - フォローボタン（自分以外のユーザーに表示）
  - 投稿一覧タブ
  - フォロワー一覧タブ
  - フォロー中一覧タブ
- 訪問済み・行きたいリストはマイページに表示（公開プロフィールには表示しない）

### 3.2 プロフィール編集

- URL：`/settings`
- 自分のプロフィールのみ編集可能
- 編集可能項目：名前（name）・自己紹介（bio）
- `PUT /api/users/[id]` で更新する

### 3.3 プロフィール画像アップロード

- 設定画面（`/settings`）から画像を選択してアップロードする
- `POST /api/upload/avatar` で Cloudinary にアップロードし URL を取得
- `PUT /api/users/[id]` で `image` URL を更新する
- 対応フォーマット：JPEG / PNG / WebP
- 最大サイズ：未定（実装時に決定）

---

## 4. 使用 API

| Method | エンドポイント | 説明 | 認証 |
|--------|--------------|------|------|
| GET | `/api/users/[id]` | ユーザー情報取得 | 不要 |
| PUT | `/api/users/[id]` | プロフィール編集 | 必要（本人のみ） |
| POST | `/api/upload/avatar` | プロフィール画像を Cloudinary にアップロード | 必要 |

**レスポンス例（ユーザー情報）**

```json
{
  "id": "cm_xxx",
  "name": "田中花子",
  "image": "https://res.cloudinary.com/...",
  "bio": "旅が好きです",
  "followerCount": 12,
  "followingCount": 8,
  "followedByCurrentUser": true,
  "postCount": 25
}
```

---

## 5. バリデーション仕様

| 項目 | フロントエンド | バックエンド |
|------|-------------|------------|
| 名前（name） | 必須・1〜50文字 | 必須・1〜100文字 |
| 自己紹介（bio） | 任意・最大文字数未定 | 任意・最大200文字 |
| プロフィール画像 | 任意・フォーマットチェック | 任意・MIME タイプ・サイズ検証 |
| 他人のプロフィール編集 | - | 403 を返す |

---

## 6. 制約・前提条件

- プロフィール編集・画像アップロードはログインユーザーのみ、かつ本人のみ可能
- プロフィール表示は未認証ユーザーも閲覧可能（未定：実装時に確認）
- プロフィール画像の最大サイズは未定

---

## 7. 対象外仕様

| 機能 | 理由 |
|------|------|
| ユーザー名（@username）の設定 | スコープ外（name のみ管理） |
| アカウント削除 | スコープ外 |
| メールアドレス変更 | スコープ外 |
| パスワード変更 | スコープ外 |
