## 1. 画面一覧

| 画面ID | 画面名 | URL | 認証要否 |
|--------|--------|-----|---------|
| S-01 | ログイン | `/login` | 不要 |
| S-02 | 新規登録 | `/signup` | 不要 |
| S-03 | 探索（全体フィード） | `/` | 不要（未認証でも閲覧可） |
| S-04 | 投稿詳細 | `/posts/[id]` | 不要（閲覧のみ） |
| S-05 | 投稿作成 | `/posts/new` | 必要 |

| S-08 | エリアタグ絞り込み | `/tags/[tag]` | 不要 |
| S-09 | 設定（プロフィール編集） | `/settings` | 必要 |
| S-10 | フォロー中フィード | `/home` | 必要 |
| S-11 | ユーザー検索 | `/search` | 不要 |

## 2. 共通レイアウト（認証済み画面）

### デスクトップ（1024px 以上）

```
+------------------+--------------------------------+
|   サイドバー      |         メインコンテンツ        |
|                  |                                |
|  [ロゴ]          |                                |
|                  |                                |
|  ナビメニュー    |                                |
|  - ホーム        |                                |
|  - 投稿作成      |                                |
|  - プロフィール  |                                |
|  - 設定          |                                |
|                  |                                |
|  [ユーザー情報]  |                                |
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| ロゴ | TripDiary のロゴ / テキスト |
| ナビメニュー | 探索・フォロー中・投稿作成・プロフィール・設定・ログアウト（未認証時は探索のみ） |
| ユーザー情報 | アバター・名前 |

### タブレット（768px 〜 1023px）

```
+--------+---------------------------+
|  縮小  |      メインコンテンツ      |
| サイド |                           |
| バー   |                           |
| （ア   |                           |
| イコ   |                           |
| ンのみ）|                           |
+--------+---------------------------+
```

### スマートフォン（767px 以下）

```
+------------------------------+
|         メインコンテンツ      |
|                              |
|                              |
+------------------------------+
|  [ホーム][投稿][プロフ][設定]  |
+------------------------------+
```

| 要素 | 内容 |
|------|------|
| ボトムナビ | 探索・フォロー中・投稿作成・プロフィール の4タブ（未認証時は探索・ログインの2タブ） |

## 3. 各画面仕様

### S-01 ログイン画面

**URL：** `/login`

```
+----------------------------+
|        TripDiary           |
|                            |
|   [メールアドレス入力]      |
|   [パスワード入力]          |
|                            |
|   [ログインボタン]          |
|                            |
|   アカウントをお持ちでない方 |
|   → [新規登録]             |
+----------------------------+
```

| 要素 | 内容 |
|------|------|
| メールアドレス | テキスト入力（type=email） |
| パスワード | パスワード入力（type=password） |
| ログインボタン | Auth.js `signIn('credentials', ...)` を呼び出す |
| 新規登録リンク | `/signup` に遷移 |

---

### S-02 新規登録画面

**URL：** `/signup`

```
+----------------------------+
|        TripDiary           |
|                            |
|   [名前入力]               |
|   [メールアドレス入力]      |
|   [パスワード入力]          |
|   [パスワード確認入力]      |
|                            |
|   [登録ボタン]              |
|                            |
|   すでにアカウントをお持ちの方|
|   → [ログイン]             |
+----------------------------+
```

---

### S-03 探索（全体フィード）画面

**URL：** `/`（未認証でも閲覧可）

```
+------------------+--------------------------------+
|   サイドバー      |  探索                          |
|                  |                                |
|                  |  [新着順▼] [カテゴリ▼] [エリア▼]|
|                  |                                |
|                  |  +----------------------------+|
|                  |  | [アバター] 田中花子          ||
|                  |  | 嵐山 竹林の道               ||
|                  |  | [写真サムネイル]             ||
|                  |  | #京都  観光スポット  ★★★★★ ||
|                  |  | 訪問日：2026年3月           ||
|                  |  | [♡ 5] [💬 2] [行きたい]    ||
|                  |  +----------------------------+|
|                  |                                |
|                  |  +----------------------------+|
|                  |  | （投稿カード繰り返し）        ||
|                  |  +----------------------------+|
|                  |                                |
|                  |  [さらに読み込む] （カーソルPN）|
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| 投稿カード | アバター・名前・スポット名・先頭写真・エリアタグ・カテゴリ・評価・訪問日・いいね数・コメント数 |
| ソート | 新着順（createdAt DESC）/ 人気順（likeCount DESC）切り替え |
| フィルター | カテゴリドロップダウン・エリアタグドロップダウン（複合絞り込み可） |
| ページネーション | カーソルベース（「さらに読み込む」ボタン） |
| 対象 | 全ユーザーの投稿（未認証でも表示） |

### S-10 フォロー中フィード画面

**URL：** `/home`（認証必須）

```
+------------------+--------------------------------+
|   サイドバー      |  フォロー中                    |
|                  |                                |
|                  |  +----------------------------+|
|                  |  | [アバター] 田中花子          ||
|                  |  | 嵐山 竹林の道               ||
|                  |  | [写真サムネイル]             ||
|                  |  | #京都  観光スポット  ★★★★★ ||
|                  |  | [♡ 5] [💬 2]               ||
|                  |  +----------------------------+|
|                  |  （フォロー中0人の場合）         |
|                  |  +----------------------------+|
|                  |  | 誰もフォローしていません。    ||
|                  |  | [探索ページへ] を使って      ||
|                  |  | 気になるユーザーを探しましょう||
|                  |  +----------------------------+|
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| 投稿カード | S-03 と同形式 |
| 表示順 | 新着順（createdAt DESC） |
| 対象 | フォロー中ユーザーの投稿のみ |
| 空状態 | フォロー0人の場合に案内メッセージと探索ページへのリンクを表示 |

### S-11 ユーザー検索画面

```
+------------------+--------------------------------+
|   サイドバー      |  ユーザーを探す                |
|                  |                                |
|                  |  [🔍 ユーザー名で検索...]       |
|                  |                                |
|                  |  +----------------------------+|
|                  |  | [アバター]  田中花子         ||
|                  |  |            旅と写真が好き    ||
|                  |  |          [フォローする]      ||
|                  |  +----------------------------+|

```

| 要素 | 内容 |
|------|------|
| 検索フォーム | ユーザー名の部分一致検索（`/api/users/search?q=`） |
| ユーザーカード | アバター・名前・自己紹介・フォローボタン（認証済みのみ） |
