## 3. 各画面仕様

### S-01 ログイン画面

**URL：** `/login`

```
+----------------------------+
|  ✈️ TripDiary              |
|  旅のスポットを記録・共有しよう |
|                            |
|   [メールアドレス入力]      |
|   [パスワード入力]  [👁]    |
|                            |
|   [ログインボタン]          |
|                            |
|   アカウントをお持ちでない方 |
|   → [新規登録]             |
+----------------------------+
```

| 要素 | 内容 |
|------|------|
| ロゴ・サブタイトル | 「✈️ TripDiary」と「旅のスポットを記録・共有しよう」をカード上部中央に表示 |
| メールアドレス | テキスト入力（type=email） |
| パスワード | パスワード入力（type=password）。右端にパスワード表示トグルボタン（👁）を配置。押している間だけ平文表示 |
| ログインボタン | Auth.js `signIn('credentials', ...)` を呼び出す |
| 新規登録リンク | カード下部に配置。`/signup` に遷移 |

**URL：** `/signup`

```
+----------------------------+
|  ✈️ TripDiary              |
|  アカウントを作成して旅を記録しよう |
|                            |
|   [ニックネーム入力]         |
|   [メールアドレス入力]      |
|   [パスワード入力]          |
|   [パスワード確認入力]      |
|                            |
|   [登録するボタン]          |
|                            |
|   すでにアカウントをお持ちの方|
|   → [ログイン]             |
+----------------------------+
```

| 要素 | 内容 |
|------|------|
| ロゴ・サブタイトル | 「✈️ TripDiary」と「アカウントを作成して旅を記録しよう」をカード上部中央に表示 |
| ニックネーム | テキスト入力。placeholder「例：旅人さくら」 |
| メールアドレス | テキスト入力（type=email） |
| パスワード | パスワード入力。placeholder「8文字以上」 |
| パスワード（確認） | パスワード入力。placeholder「もう一度入力してください」 |
| 登録するボタン | `POST /api/auth/signup` → 自動ログイン → `/dashboard` へ遷移 |
| ログインリンク | カード下部に配置。`/login` に遷移 |

### S-03 探索（全体フィード）画面

**URL：** `/`（未認証でも閲覧可）

```
+------------------+--------------------------------+
|   サイドバー      |  ❤️ 人気の旅スポット            |
|                  |  [投稿カードグリッド 3列]        |
|                  |                                |
|                  |  📍 エリアから探す              |
|                  |  [エリアタグカード一覧]          |
|                  |  （右上）都道府県で探す →        |
|                  |                                |
|                  |  🗂️ カテゴリから探す            |
|                  |  [カテゴリカード一覧]            |
|                  |                                |
|                  |  ⭐ カテゴリ別高評価スポット     |
|                  |  [カテゴリ×最高評価投稿カード]   |
|                  |                                |
|                  |  🆕 新着の旅スポット            |
|                  |  [投稿カードグリッド 3列]        |
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| 人気の旅スポット | いいね数降順で上位6件をカードグリッド表示。「もっと見る」リンクで検索（人気順）へ遷移 |
| エリアから探す | 投稿に付いたエリアタグをサムネイル付きカードで一覧表示。クリックでエリアタグ絞り込み画面へ。右上に「都道府県で探す →」リンクを配置し、検索のエリアタブ（`/search?tab=area`）へ遷移する |
| カテゴリから探す | カテゴリ別にアイコン・件数付きカードで表示。クリックでカテゴリ絞り込み |
| カテゴリ別高評価スポット | 各カテゴリの中で評価（rating）が最も高い投稿を1件ずつカード表示。同評価の場合はいいね数降順。「人気の旅スポット」に含まれる投稿は除外 |
| 新着の旅スポット | 新着順（createdAt DESC）で上位6件をカードグリッド表示。「もっと見る」リンクで検索（新着順）へ遷移 |
| 対象 | 全ユーザーの投稿（未認証でも表示） |

### S-10 マイページ

**URL：** `/mypage`（認証必須）

ページ上部に「🔒 この画面は自分のみ表示」バナーを表示。タブ切り替えで各コンテンツを表示する。

```
+------------------+--------------------------------+
|   サイドバー      |  [タイトル]  🔒この画面は自分のみ表示|
|                  |                                |
|                  |  [旅行プラン][自分の投稿][旅行レポート][行きたい][訪問済み][フォロー中の投稿] |
|                  |                                |
|                  |  （選択タブのコンテンツ）        |
+------------------+--------------------------------+
```

| タブ | 内容 |
|------|------|
| 🗺️ 旅行プラン | 進行中プランのカード一覧 + 完了済みプランのアコーディオン。カウントバッジは完了済みを除外。デフォルト表示タブ |
| ✈️ 自分の投稿 | 自分が投稿したスポットのカードグリッド。費用・費用内訳を表示。各カードにホバーで ✏️（編集）・🗑️（削除）アイコンを表示 |
| 📋 旅行レポート | 年別の旅まとめカード・カテゴリグラフ・月別費用グラフ・訪問都道府県一覧・年別タイムラインを表示。詳細は S-17 参照 |
| 🔖 行きたい | 行きたい登録した投稿カードグリッド |
| ✅ 訪問済み | 訪問済み登録した投稿カードグリッド |
| 👥 フォロー中の投稿 | フォロー中ユーザーの投稿カードグリッド（新着順） |

### S-13〜16 旅行プラン画面

**URL：** S-14 `/plans/[id]`・S-15 `/plans/new`・S-16 `/plans/[id]/edit`（すべて認証必須）

**S-14 旅行プラン詳細**

```
+------------------+--------------------------------+
|   サイドバー      |  🗺️ プランタイトル              |
|                  |  📅 出発日 〜 帰着日            |
|                  |  📝 メモ                        |
|                  |  💴 予算：XX,XXX円（内訳タグ）   |
|                  |                                |
|                  |  [スポット一覧]                 |
|                  |  1. スポット名 [この旅を記録する→]|
|                  |  2. スポット名 [この旅を記録する→]|
|                  |                                |
|                  |  [このプランから作成した投稿]     |
|                  |                                |
|                  |  [旅行を完了済みにする] / [↩完了を取り消す] |
|                  |  [✏️ 編集] [削除]               |
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| スポット一覧 | 追加済みスポットを番号付きで表示。「この旅を記録する」ボタンで投稿作成画面へ遷移（タイトル・エリアタグ等プリセット） |
| リンク済み投稿 | planId が一致する投稿を一覧表示 |
| 予算内訳 | 項目ごとにタグ形式で表示（合計金額も表示） |
| 完了操作 | 未完了時：「旅行を完了済みにする」ボタン。完了時：「↩ 完了を取り消す」ボタン |

**S-15 旅行プラン作成 / S-16 旅行プラン編集**

| 入力項目 | 内容 |
|---------|------|
| タイトル | 必須・最大60文字 |
| 出発日・帰着日 | 任意・日付入力（終了日 ≥ 開始日） |
| 予算 | 合計金額を自動計算。内訳を ＋ボタンで項目追加（金額 + ラベルの行） |
| メモ | 任意・自由テキスト |
| スポット | 行きたいリストから優先表示。番号付き選択済みセクション + 候補リスト（その他はアコーディオン） |

### S-11 総合検索画面

```
+------------------+--------------------------------+
|   サイドバー      |  🔍 検索                       |
|                  |                                |
|                  |  [スポット名・エリア・ユーザー名で検索...] |
|                  |                                |
|                  |  [旅スポット][エリア][ユーザー]  |
|                  |                                |
|                  |  [全て][観光][グルメ][宿・ホテル]...  |
```

### S-12 通知画面

**URL：** `/notification`（認証必須）

```
+------------------+--------------------------------+
|   サイドバー      |  🔔 通知                       |
|                  |                                |
|                  |  [アバター] 田中花子 が いいねしました  |
|                  |  [アバター] 田中花子 がコメントしました |
|                  |  [アバター] 田中花子 がフォローしました |
|                  |  ...                           |
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| いいね通知 | 自分の投稿にいいねされた際に表示。投稿詳細へリンク |
| コメント通知 | 自分の投稿にコメントされた際に表示。コメント本文の冒頭も表示。投稿詳細へリンク |
| フォロー通知 | 自分がフォローされた際に表示。フォローしたユーザーのプロフィールへリンク |
| 未読バッジ | サイドバー・ボトムナビの 🔔 アイコンに未読件数を赤バッジで表示 |
| 既読処理 | 通知画面を開いた時点で全件既読にする |

### S-17 旅行レポート画面

**URL：** `/mypage?tab=report`（認証必須）

```
+------------------+--------------------------------+
|   サイドバー      |  📋 旅行レポート                |
|                  |                                |
|                  |  [年選択ドロップダウン▼]         |
|                  |                                |
|                  |  ✨ XXXX年の旅まとめ            |
|                  |  [カード7枚グリッド]             |
|                  |   旅行回数 / 訪問都道府県数      |
|                  |   訪問スポット数 / 投稿数        |
|                  |   写真枚数 / 費用合計 / 最多エリア|
|                  |                                |
|                  |  📊 カテゴリ別投稿数（バーグラフ）|
|                  |  💴 月別旅行費用（棒グラフ）     |
|                  |  📍 訪問した都道府県（タグ一覧）  |
|                  |                                |
|                  |  🗓️ 年別タイムライン            |
|                  |  2026年の旅（N件）              |
|                  |   [サムネイル] スポット名 日付   |
|                  |   ▼ 残りN件を表示              |
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| 年選択 | 自分の投稿がある年をドロップダウンで選択。切り替えで全セクション更新 |
| 年間まとめカード | Spotify Wrapped 風にグラデーションカードを順番にフェードインアニメーション表示 |
| カテゴリ別バーグラフ | 選択年の投稿をカテゴリ別に集計し CSS バーグラフで表示 |
| 月別費用棒グラフ | 選択年の月別 `cost` 合計を CSS 縦棒グラフで表示 |
| 訪問都道府県タグ | 選択年の `prefecture` 一覧をバッジ形式で表示 |
| 年別タイムライン | 全年の投稿を年別セクションに分けてリスト表示。各年3件まで表示し「▼ 残りN件を表示」でトグル展開 |

**集計ロジック（旅行回数）**
`completed = true` かつ `startDate` が選択年に一致する Plan の件数を旅行回数として表示。
