## 2. 共通レイアウト（認証済み画面）

### デスクトップ（1024px 以上）

```
+------------------+--------------------------------+
|   サイドバー      |         メインコンテンツ        |
|                  |                                |
|  [ロゴ]          |                                |
|                  |                                |
|  ナビメニュー    |                                |
|  - ホーム        |                                |
|  - 投稿作成      |                                |
|  - プロフィール  |                                |
|  - 設定          |                                |
|                  |                                |
|  [ユーザー情報]  |                                |
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| ロゴ | TripDiary のロゴ / テキスト |
| ナビメニュー（上部） | ホーム・検索・投稿する（認証済み時のみ）・区切り線・管理画面・公開プロフィール（認証済み時のみ） |
| ユーザー情報（下部） | アバター・名前・設定リンク・ログアウトボタン（未認証時はログイン/新規登録ボタン） |

### タブレット（768px 〜 1023px）

```
+--------+---------------------------+
|  縮小  |      メインコンテンツ      |
| サイド |                           |
| バー   |                           |
| （ア   |                           |
| イコ   |                           |
| ンのみ）|                           |
+--------+---------------------------+
```

### スマートフォン（767px 以下）

```
+------------------------------+
|         メインコンテンツ      |
|                              |
|                              |
+------------------------------+
|  [ホーム][投稿][プロフ][設定]  |
+------------------------------+
```

| 要素 | 内容 |
|------|------|
| ボトムナビ | ホーム・検索・投稿・マイページ の4タブ（未認証時はホーム・検索・ログインの3タブ） |

## 3. 各画面仕様

### S-01 ログイン画面

**URL：** `/login`

```
+----------------------------+
|        TripDiary           |
|                            |
|   [メールアドレス入力]      |
|   [パスワード入力]          |
|                            |
|   [ログインボタン]          |
|                            |
|   アカウントをお持ちでない方 |
|   → [新規登録]             |
+----------------------------+
```

| 要素 | 内容 |
|------|------|
| メールアドレス | テキスト入力（type=email） |
| パスワード | パスワード入力（type=password） |
| ログインボタン | Auth.js `signIn('credentials', ...)` を呼び出す |
| 新規登録リンク | `/signup` に遷移 |

### S-03 ホーム（ポータル）画面

**URL：** `/`（未認証でも閲覧可）

```
+------------------+--------------------------------+
|   サイドバー      |  ❤️ 人気の旅スポット            |
|                  |  [投稿カードグリッド 3列]        |
|                  |                                |
|                  |  📍 エリアから探す              |
|                  |  [エリアタグカード一覧]          |
|                  |                                |
|                  |  🗂️ カテゴリから探す            |
|                  |  [カテゴリカード一覧]            |
|                  |                                |
|                  |  🆕 最新の旅スポット            |
|                  |  [投稿カードグリッド 3列]        |
|                  |                                |
|                  |  🏆 トラベラーランキング        |
|                  |  [上位3名・フォローボタン付き]   |
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| 人気の旅スポット | いいね数降順で上位6件をカードグリッド表示 |
| エリアから探す | 投稿に付いたエリアタグをサムネイル付きカードで一覧表示。クリックでエリアタグ絞り込み画面へ |
| カテゴリから探す | カテゴリ別にアイコン・件数付きカードで表示。クリックで絞り込み |
| 最新の旅スポット | 新着順（createdAt DESC）で上位6件をカードグリッド表示 |
| トラベラーランキング | TabiScore 上位3名をランクバッジ付きで表示。フォローボタンあり |
| 対象 | 全ユーザーの投稿（未認証でも表示） |

### S-10 マイページ（管理画面）

**URL：** `/mypage`（認証必須）

```
+------------------+--------------------------------+
|   サイドバー      |  🔒 管理画面 [公開プロフィールを見る →]|
|                  |                                |
|                  |  [アバター] 田中さき            |
|                  |  TabiScore ✈️ 120 pt           |
|                  |  投稿 5 / フォロワー 3 / フォロー中 2 / 行きたい 4 / 訪問済み 2 |
|                  |  [プロフィールを見る] [設定]     |
|                  |                                |
|                  |  [タイムライン][フォロー中の投稿][🔖行きたい][✅訪問済み][✈️自分の投稿] |
|                  |                                |
|                  |  （選択タブのコンテンツ）        |
+------------------+--------------------------------+
```

| タブ | 内容 |
|------|------|
| タイムライン | フォロー中ユーザーの投稿・いいね・コメント・フォローのアクティビティ一覧 |
| フォロー中の投稿 | フォロー中ユーザーの投稿カードグリッド（新着順） |
| 🔖 行きたい | 行きたい登録した投稿カードグリッド |
| ✅ 訪問済み | 訪問済み登録した投稿カードグリッド |
| ✈️ 自分の投稿 | 自分が投稿したスポットのカードグリッド |

### S-11 総合検索画面

**URL：** `/search`（未認証でも閲覧可）

```
+------------------+--------------------------------+
|   サイドバー      |  🔍 検索                       |
|                  |                                |
|                  |  [スポット名・エリア・ユーザー名で検索...] |
|                  |                                |
|                  |  [すべて][旅スポット][ユーザー][🏆ランキング] |
|                  |                                |
|                  |  （検索結果・タブ別コンテンツ）  |
+------------------+--------------------------------+
```

| 要素 | 内容 |
|------|------|
| 検索フォーム | スポット名・エリアタグ・カテゴリ・ユーザー名の横断検索（リアルタイムフィルタ） |
| すべてタブ | 旅スポット + ユーザーの横断結果（未入力時は最新投稿6件を表示） |
| 旅スポットタブ | 条件に合う投稿カードグリッド |
| ユーザータブ | 条件に合うユーザー一覧（TabiScore・フォローボタン付き） |
| 🏆 ランキングタブ | TabiScore 降順の全ユーザー一覧（ランクバッジ付き） |
