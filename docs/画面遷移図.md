## 1. 画面遷移図

```mermaid
graph TD
    Start([アクセス]) --> Explore[S-03 探索]

    Explore -->|ログインリンク| Login[S-01 ログイン]
    Explore -->|投稿カードをクリック| PostDetail[S-04 投稿詳細]
    Explore -->|アバター / 名前をクリック| UserProfile[S-07 ユーザープロフィール]
    Explore -->|エリアタグをクリック| TagPage[S-08 エリアタグ絞り込み]
    Explore -->|フォロー中リンク（要認証）| HomeFeed[S-10 フォロー中フィード]
    Explore -->|検索アイコン| Search[S-11 ユーザー検索]

    Login -->|ログイン成功| Explore
    Login -->|新規登録リンク| Signup[S-02 新規登録]
    Signup -->|登録成功| Explore
    Signup -->|ログインリンク| Login

    HomeFeed -->|投稿カードをクリック| PostDetail
    HomeFeed -->|投稿作成ボタン| PostNew[S-05 投稿作成]
    HomeFeed -->|アバター / 名前をクリック| UserProfile
    HomeFeed -->|エリアタグをクリック| TagPage
    HomeFeed -->|探索リンク| Explore

    PostDetail -->|ブラウザ戻るボタン| PrevPage[遷移元の画面]
    PostDetail -->|編集ボタン（本人のみ）| PostEdit[S-06 投稿編集]
    PostDetail -->|削除ボタン（本人のみ）→ 確認後| PrevPage
    PostDetail -->|アバター / 名前をクリック| UserProfile
    PostDetail -->|エリアタグをクリック| TagPage

    PostNew -->|投稿成功| PostDetail
    PostNew -->|キャンセル| PrevPage

    PostEdit -->|更新成功| PostDetail
    PostEdit -->|キャンセル| PostDetail

    UserProfile -->|投稿カードをクリック| PostDetail
    UserProfile -->|エリアタグをクリック| TagPage
    UserProfile -->|設定リンク（自分のみ）| Settings[S-09 設定]

    TagPage -->|投稿カードをクリック| PostDetail

    Settings -->|保存成功| UserProfile
    Settings -->|キャンセル| UserProfile
    Settings -->|ログアウト| Explore

    Search -->|ユーザーカードをクリック| UserProfile
```

## 2. 認証ガードの方針

| URL パターン | 未認証時の挙動 |
|------------|-------------|
| `/` | そのまま表示（探索ページは公開） |
| `/posts/[id]` | そのまま表示（閲覧のみ） |
| `/users/[id]` | そのまま表示（閲覧のみ） |
| `/tags/[tag]` | そのまま表示 |
| `/search` | そのまま表示 |
| `/home` | `/login` にリダイレクト |
| `/posts/new` | `/login` にリダイレクト |
| `/posts/[id]/edit` | `/login` にリダイレクト |
| `/settings` | `/login` にリダイレクト |
